CounterExample(Func)

Adds custom debug information that will be displayed when a property test fails. Use this to include variable values or other context to help diagnose failures.

Example:

 var property =
     from x in Gen.Int32(Range.Constant(0, 1000)).ForAll()
     from y in Gen.Int32(Range.Constant(-100, 1000)).ForAll()
     from _ in Property.CounterExample(() => $"x = {x}, y = {y}, x * y = {x * y}")
     select x * y >= 0;
