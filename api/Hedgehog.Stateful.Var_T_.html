Symbolic variable referencing a command's output. Symbolic variables are placeholders that let us chain commands by using one command's result as input to another, even before execution. A symbolic variable is not yet bound to a generated value and is used to represent a value in the model before binding occurs.

type Var<'T> =
  member Resolve : Env -> 'T
  member ResolveOr : env:Env * fallback:'T -> 'T
